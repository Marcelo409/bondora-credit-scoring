# Data

This project uses Bondora's public P2P loan dataset.

## How to obtain the dataset

1. Go to: https://bondora.com/en/public-statistics
2. Download the "investor" version of the loan dataset (XLSX format, ~150 MB)
3. Place the file in this folder as `loan_dataset_investor.xlsx`
4. Update the file path in cell 2 of the notebook if needed

## File structure expected

- `loan_dataset_investor.xlsx` — raw data, ~737k loans, 31 columns

The dataset is not included in this repository due to size constraints
and to respect Bondora's distribution preferences. It is publicly and
freely available from the link above.
